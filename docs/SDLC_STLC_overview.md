# SDLC & STLC Overview
Author: dhimanrgv | Date: 2021-05-18

## SDLC Phases
| Phase | QA Involvement |
|---|---|
| Requirements | Review, raise ambiguities, start RTM |
| Design | Review for testability |
| Implementation | Prepare test cases |
| Testing | Execute, log defects, regression |
| Deployment | Smoke testing, sign-off |
| Maintenance | Regression for each patch |

## STLC Phases
1. Requirement Analysis -- identify testable requirements, prepare RTM
2. Test Planning -- scope, effort estimation, tools selection
3. Test Case Design -- positive, negative, edge cases
4. Environment Setup -- CI/CD, browsers, test data
5. Test Execution -- manual + automation, defect logging
6. Test Closure -- summary report, metrics, retrospective

## Testing Types
- Smoke, Regression, Functional, Integration, System, UAT
- Black Box, White Box, Negative Testing, API Testing

## Defect Lifecycle
New -> Assigned -> In Progress -> Fixed -> Re-test -> Closed
